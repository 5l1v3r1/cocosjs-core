## Cocosjs Core

It is used for dapp.
Use it to hook up with various clients, including chrome plugins, wallet clients, IOS wallets, Android wallets.

## How to use

npm i cocosjs-core

more information, visit https://www.npmjs.com/package/cocosjs-core
